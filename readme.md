# Yasin Unal - Personal Portfolio

## Overview
This repository hosts the personal portfolio website of Yasin Unal, a Computer Engineer and Researcher specializing in Electric Vehicle Routing, Optimization, and Artificial Intelligence. The portfolio showcases academic background, professional experience, publications, and technical projects.


## Technologies Used
- HTML5
- CSS3 (Custom styling + Bootstrap 5.3.3)
- JavaScript (DOM manipulation, LocalStorage for language preference)

## Project Structure
- `index.html`: Main curriculum vitae page.
- `projects.html`: Detailed showcase of R&D and personal projects.
- `certificates.html`: Comprehensive list of certifications.
- `assets/`: Contains static assets including stylesheets.

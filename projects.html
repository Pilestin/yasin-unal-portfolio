Routing Markup Language (RML)

2024 - Present

Standardized data format for vehicle routing problems to enable interoperability between solvers

Components:
  • XML/JSON schema for vehicle specifications, constraints, and capabilities
  • Environment modeling including road networks, charging stations, and depots
  • Validation toolkit and parser library for Python and JavaScript
  • Integration with routing solvers (ALNS, SA, TS)
XML/JSON Schema Design
